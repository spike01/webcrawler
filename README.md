# Web crawler

### Description

A simple webcrawler. In Go.

### Installing

Install deps:
```
go get ./...
```

Build:
```
go build
```

Run:
```
./webcrawler http://yourdomain.here
```

Run without building:
```
go run main.go http://yourdomain.here
```

Available flags:
```
  -v    make the operation more talkative
```

### Running the tests
```
go test ./...
```
